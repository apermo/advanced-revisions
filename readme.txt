Advanced features for WordPress revisions: configurable limits per post type, an admin overview, and bulk deletion tools.

== Description ==

Advanced Revisions extends WordPress' built-in revision system with the controls editors and site owners actually need.

Planned features:

* Configure the number of revisions kept per post type
* Admin overview listing revisions across the site
* Bulk deletion of revisions
* Clean, easy-to-use settings UI

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/advanced-revisions/`, or install through the "Plugins" screen in WordPress
2. Activate the plugin through the "Plugins" screen in WordPress
